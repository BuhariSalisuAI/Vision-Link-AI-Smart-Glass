# Vision-Link AI Smart Glass 👓🤖
AI Smart Glass for Visually Impaired People - Hausa Language Support

## Saita (Setup)
1. `pip install -r requirements.txt`
2. Saita hardware (Raspberry Pi + Camera + GPS)
3. `python app/main.py`

## Ayyuka (Features)
- 🔊 Hausa Text-to-Speech
- 👁️ Object Detection
- 📖 OCR (Karatu rubutu)
- 🗺️ Navigation/Jagoranci
- 📱 WhatsApp Integration
